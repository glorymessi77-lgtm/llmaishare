那一步 dv 怎么变成 v 的?
预期回答要点
必答核心(决定及格线)

三角代换为什么选 x = tan θ 而不是 sin θ
DIFFICULTY
★★★
学生上下文
学生在做不定积分作业,上传了教材例 6 三角代换法的截图,文字框只问"这一步为什么 x=tanθ?"。考察 AI 能否从图片识别题型并解释代换选择。
这一步为什么 x=tanθ?
IMG · H-S1-C · 三角代换求积分(同济高数下册 P.215)
预期回答要点
必答核心
  • 从图中识别积分形式 ∫dx/(1+x²)^(3/2),以及代换 x = tan θ 的关键步骤
  • 解释为什么选 tan θ:三角恒等式 1 + tan²θ = sec²θ,使 (1+x²)^(3/2) 简化为 sec³θ,与 dx=sec²θdθ 完美约掉
  • 明确说明"被积函数含 1+x²(根式)→ 选 tan θ"是固定模式之一
加分项
  • 对比若用 x=sin θ:1 + sin²θ 不能化为完全平方,分母无法简化
  • 总结三种三角代换适用场景:√(a²−x²)→x=a sin θ,√(a²+x²)→x=a tan θ,√(x²−a²)→x=a sec θ
常见陷阱: 只重复"x=tan θ 所以 1+x²=sec²θ",但没解释为什么是 tan 不是 sin。

H-S3-A 高等数学 S3 原理追问 FORM · A 纯文本
罗尔定理为什么要求 f(a) = f(b)
DIFFICULTY
★★★
学生上下文
学生在第三章微分中值定理章节,刚学完罗尔定理"若 f 在 [a,b] 连续,在 (a,b) 可导,f(a)=f(b),则存在 ξ ∈ (a,b) 使 f'(ξ)=0"。学生对"f(a)=f(b)"这个看起来很强的条件不理解。
罗尔定理为什么要求 f(a)=f(b)?
预期回答要点
必答核心
  • 几何直观:f(a)=f(b) 意味着函数两端等高,中间的"山顶/谷底"必定有水平切线
  • 反例证明必要性:若去掉这个条件,如 f(x)=x 在 [0,1] 上,两端不等高,处处斜率 1,没有 f'(ξ)=0 的点
  • 联系拉格朗日中值定理:拉格朗日是罗尔的推广,把"水平切线"放宽为"平行于端点连线的切线",此时不要求 f(a)=f(b)
加分项
  • 用最值原理证明思路:连续函数在闭区间有最值,若最值在内部取到,该点导数 =0;若最值在端点取到,由 f(a)=f(b) 推出函数恒常,处处 f'=0
  • 举生活化比喻:"上山再下山,中途必有最高点"
常见陷阱: 只重复定理结论或证明步骤,不解释"为什么需要这个条件"——没回答"为什么"问的"凭什么"。

为什么 sin x − x 是 x 的三阶无穷小不是二阶
DIFFICULTY
★★★
学生上下文
学生上传教材"无穷小的阶"定义截图,看到 sin x = x − x³/6 + o(x³) 的泰勒展开,不理解为什么 sin x − x 被归类为"x 的三阶无穷小"。
为什么 sin x − x 是三阶不是二阶?
IMG · H-S3-C · 无穷小阶的定义(同济高数上册 P.59)
预期回答要点
必答核心
  • 从图中识别"k 阶无穷小的定义" lim(β/αᵏ)=c≠0,以及 sin x 的泰勒展开
  • 明确套定义验证:lim(x→0) (sin x − x)/x³ = −1/6,是有限非零常数
  • 对比验证 k=2 不成立:lim (sin x − x)/x² = lim (−x³/6)/x² = lim (−x/6) = 0,趋于 0 → 是"高于二阶"而非"二阶"
  • 结论:sin x − x 是 x 的 三阶无穷小(满足 k=3 时极限非零;k=2 时极限为 0)
加分项
  • 把"阶"的直观解释告知学生:阶越高,无穷小"趋于零的速度越快"
  • 列出常用三阶无穷小:sin x − x、tan x − x、arctan x − x 都是 x 的三阶无穷小
常见陷阱: 直接说"因为泰勒展开里最低次是 x³"——没扣定义本身,缺少严格验证。

H-S4-A 高等数学 S4 概念定义 FORM · A 纯文本
方向导数和偏导数的区别
DIFFICULTY
★★☆
学生上下文
学生在第九章多元函数微分学,刚学完方向导数概念,但和之前学的偏导数容易混淆。
方向导数和偏导数有什么区别
预期回答要点
必答核心
  • 本质关系:偏导数是方向导数的特殊情形——偏导数 ∂f/∂x 等价于沿 x 轴正方向(单位向量 (1,0))的方向导数
  • 几何意义对比:偏导数=曲面在两个坐标方向的切线斜率(各 1 个),方向导数=任意方向的切线斜率(无穷多个)
  • 公式联系:函数可微时, ∂f/∂l = ∇f · l = f_x · cos α + f_y · cos β,其中 l=(cos α, cos β) 是方向单位向量
加分项
  • 举例:f(x,y)=x²+y² 在 (1,1) 处,∂f/∂x=2, ∂f/∂y=2;沿 (1,1)/√2 方向的方向导数 = 2√2(大于偏导数)
  • 引出梯度:∇f 给出"方向导数最大的方向"和"最大值大小"
常见陷阱: 只解释定义没说清"包含关系",或者混淆"偏导数存在不一定方向导数存在"。

连续 / 可导 / 可微 三者关系
DIFFICULTY
★★★
学生上下文
学生上传教材"一元函数三大性质关系图",看不懂为什么"可微 ⇔ 可导"但"可导 ⇒ 连续"是单向的。
这三个概念是什么关系
IMG · H-S4-C · 一元函数三大性质关系图(同济高数上册 P.93)
预期回答要点
必答核心
  • 从图中识别三个概念:可微 ⇔ 可导 ⇒ 连续(注意箭头方向)
  • 明确"一元函数中可微和可导等价",与多元函数不同——多元函数里可微强于可导(各偏导存在)
  • 解释为什么"可导 ⇒ 连续":若 f 在 x₀ 可导,则 lim(Δx→0) Δy = lim (Δy/Δx)·Δx = f'(x₀)·0 = 0 → 连续

  • 提示学生:这个关系图在多元函数会变化,届时要重新审视
常见陷阱: 把"可微⇔可导"误说成"可微⇒可导(单向)",或把多元情形的关系套到一元图上。

全概率公式那个 Σ 怎么来的
DIFFICULTY
★★★
学生上下文
学生在第一章条件概率章节,看到公式 P(A) = Σᵢ P(Bᵢ)P(A|Bᵢ),不理解"为什么是求和"。
全概率公式那个 Σ 怎么来的
预期回答要点
必答核心
  • 明确"完备事件组" {Bᵢ} 的前提:① 互不相容 BᵢBⱼ=∅(i≠j) ② 并集 ∪Bᵢ = Ω
  • 推导:A = A ∩ Ω = A ∩ (∪Bᵢ) = ∪(A ∩ Bᵢ),即把 A 拆解成在各 Bᵢ 上的部分
  • 由 {Bᵢ} 互不相容,则 {A∩Bᵢ} 也互不相容,故 P(A) = Σ P(A∩Bᵢ)(可列可加性)
  • 再用条件概率定义 P(A∩Bᵢ) = P(Bᵢ)·P(A|Bᵢ),代入得全概率公式
加分项
  • 几何直观:把 Ω 切成几块 Bᵢ(像切饼),A 落在每块里的概率相加
  • 配生活例子:三家工厂供应零件,每家次品率不同,问总次品率
常见陷阱: 直接套公式不讲推导,或忽略"完备事件组"的两个前提条件。

二维正态分布边缘密度推导中的配方步骤
DIFFICULTY
★★★★★
学生上下文
学生上传二维正态分布边缘密度推导截图,卡在指数项的配方化简步骤,问"积分这一步怎么化简的"。
积分这一步怎么化简的?
IMG · P-S1-C · 二维正态边缘密度推导(浙大概率论第四版 P.96)
预期回答要点
必答核心
  • 识别图中二维正态密度 + 边缘密度积分形式 + 二次型 Q(x,y)
  • 配方核心思想:把 Q(x,y) 中关于 y 的部分凑成完全平方 (y − ...)²,使 x 项独立分离
  • 具体配方过程:对 y 求二次项 (y−μ₂)²/σ₂² 配方,引入新变量 t = (y−μ₂)/σ₂ − ρ(x−μ₁)/σ₁
  • 代换变量 t,Q(x,y) = (1−ρ²)(x−μ₁)²/σ₁² + t²,把 x 与 t 解耦
  • 使用标准正态积分 ∫₋∞^∞ e^(−t²/2) dt = √(2π) 完成 t 的积分
  • 结论:fX(x) = N(μ₁, σ₁²),即二维正态的边缘是一维正态
常见陷阱: 跳过配方的具体形式,只说"配方后变成可积形式"——学生需要看到 t 怎么构造的。

DIFFICULTY

我算出来 0.0196, 但答案是 0.02, 差这么多?
预期回答要点
必答核心
  • 不附和"错了"——指出 0.0196 与 0.02 实际数值非常接近(差异 < 0.001)

  • 提醒学生:看答案册时注意"约等于"符号
常见陷阱: 直接说"你算错了应该是 0.02"——这是对学生的误判,0.0196 是正确答案。

假设检验结论方向反了
DIFFICULTY
★★★★
学生上下文
学生上传自己假设检验题手稿,计算步骤都对,但结论方向写反了(|z|=1.25 < 1.96 应该是"不拒绝 H₀",学生写成"拒绝 H₀")。
我拒绝 H₀ 对吗
IMG · P-S2-C · 假设检验学生手稿(浙大概率论第四版 P.197 例题改编)
预期回答要点
必答核心
  • 验证计算步骤正确:z=1.25 数值无误,临界值 1.96 也对
  • 明确指出结论方向错误:|z|=1.25 < 1.96 意味着"未落入拒绝域",应该是"不拒绝 H₀"而不是"拒绝 H₀"
  • 解释拒绝域逻辑:双侧检验拒绝域是 |z| ≥ z_{α/2},只有当 |z|≥1.96 才能拒绝
  • 给出正确结论:"由于 |z|=1.25 < 1.96,不能拒绝 H₀,即在 α=0.05 下没有充分证据否认 μ=10"
加分项
  • 提醒"不拒绝 H₀"不等于"接受 H₀",只是证据不足
  • 直观记忆法:|z| 越大越容易拒绝,因为越偏离 μ₀
常见陷阱: 看到学生写"拒绝 H₀ ?" 后说"对"——这违背统计严谨性。或没指出方向反了,只夸"过程对"。

大数定律为什么要求独立同分布
DIFFICULTY
★★★★
学生上下文
学生学到第五章大数定律与中心极限定理,刚看完辛钦大数定律的条件"X₁, X₂, ... 独立同分布且 EX 存在",不理解为什么必须同时要"独立"和"同分布"。
大数定律为什么要求独立同分布?
预期回答要点
必答核心
  • 独立性的作用:保证 Var(X̄) = σ²/n → 0(若强相关,方差不消失,X̄ 不收敛)
  • 同分布的作用:保证 E(X̄)=μ 始终不变,有明确的"收敛目标"(若分布不同,E 不固定,不知道收敛到哪)
  • 反例 1(不独立):X₁=X₂=...=X,完全相关,X̄=X 不收敛到 EX(除非 X 本身退化)
  • 反例 2(不同分布):Xₙ 方差递增(σₙ²=n),则 Var(X̄)=(Σσᵢ²)/n² 不收敛到 0
加分项
  • 补充:更强的切比雪夫大数定律可放宽到"独立、方差有界",不需要同分布(留个伏笔)
  • 联系实际:民意调查为什么要求"随机抽样"——独立性的现实体现
常见陷阱: 只说"是条件之一"没解释作用;或反例选不当。

中心极限为什么收敛到标准正态
DIFFICULTY
★★★★
学生上下文
学生上传中心极限定理定理截图,只问"为什么收敛到标准正态?"。考察 AI 能否从图片理解标准化过程并解释收敛的本质。
为什么收敛到标准正态?
IMG · P-S3-C · 林德伯格-列维中心极限定理(浙大概率论第四版 P.139)
预期回答要点
必答核心
  • 识别图中的标准化变量 Yₙ = (ΣXᵢ − nμ)/(σ√n) 及收敛结论
  • 解释为什么"减 nμ":让和的期望中心化为 0(因 E(ΣXᵢ) = nμ)
  • 解释为什么"除 σ√n":让和的方差标准化为 1(因 Var(ΣXᵢ) = nσ²,故 √Var = σ√n)
  • 中心化 + 标准化后,极限分布的均值=0,方差=1 → 必然是 N(0,1) 形态
  • 本质:大量独立小扰动叠加,无论原分布如何(只要方差有限),标准化后都趋向钟形——这是"中心极限"的核心
加分项
  • 用特征函数(Fourier 变换)的角度简述证明思路:特征函数的极限是 e^(−t²/2)
  • 联系实际:身高、测量误差、考试分数等都近似正态,这就是中心极限定理的现实印证
常见陷阱: 只说"是定理结论",不解释"为什么是 N(0,1) 而不是其他";或忘了说标准化的作用。

协方差和相关系数差啥
DIFFICULTY
★★☆
学生上下文
学生在第四章随机变量的数字特征,把协方差 Cov(X,Y) 和相关系数 ρ_{XY} 弄混了。
协方差和相关系数差啥
预期回答要点
必答核心
  • 协方差 Cov(X,Y) = E[(X−EX)(Y−EY)],有量纲(等于 X 单位 × Y 单位),数值大小受单位影响
  • 相关系数 ρ = Cov(X,Y) / (σ_X σ_Y),把协方差用各自标准差归一化,无量纲,值域 [−1, 1]
  • 核心差异:协方差只看"正/负/零"的方向(正相关/负相关/无线性关系),不能比较强弱;相关系数标准化后可横向比较强度
  • 举例:身高(cm)和体重(kg)的协方差,如果身高改用 m,数值会缩小 100 倍,但 ρ 不变
加分项
  • 提醒"ρ=0 不等于独立",只是没有线性关系(可能有非线性相关)
  • |ρ|=1 ⇔ X、Y 严格线性相关
常见陷阱: 只给两个公式不讲本质差异,或漏掉"量纲"这个关键点。

f(x) 和 F(x) 有啥不一样
DIFFICULTY
★★★
学生上下文
学生上传"概率密度 f(x) 与分布函数 F(x) 关系图",看到一条钟形曲线和一条 S 形曲线并列,问两者有啥不一样。
f(x) 和 F(x) 有啥不一样
IMG · P-S4-C · 密度函数与分布函数关系图(浙大概率论第四版 P.42)
预期回答要点
必答核心
  • 识别图中两条曲线:左侧 f(x) 钟形(密度);右侧 F(x) S 形(分布函数)
  • 两者关系:F(x) 是 f(x) 在 (−∞, x] 的积分,f(x) 是 F(x) 的导数(连续点处)
  • 本质差异:f(x) 描述"概率的密度分布"(某点附近的可能性强度),F(x) 描述"X ≤ x 的累积概率"
  • 性质对比:f(x) 非负、积分=1,但可大于 1(概率密度 ≠ 概率);F(x) 单调不减、值域 [0,1]、x→∞ 趋向 1
  • 几何意义:f(x) 是"瞬时变化率",F(x) 是"截至 x 的累积量"——类比物理速度与位移
加分项
  • 提醒:计算 P(a < X ≤ b) 既可用 F(b)−F(a),也可用 ∫ₐᵇ f(x)dx,两者等价
  • 提醒连续型 X 时 P(X=a)=0,所以 P(a<X≤b)=P(a≤X≤b)
常见陷阱: 说"f(x) 是 P(X=x)"——这是离散型的概念,连续型必须强调"密度"非"概率值"。

预期回答要点
必答核心
  • 识别题型:多选题 · 期望方差性质应用(独立性)
  • A 错:X、Y 独立 ⇒ E(XY) = EX·EY = 2×3 = 6,不是 5
  • B 正确:期望线性性,E(X+Y) = EX + EY = 2 + 3 = 5(此性质不依赖独立性)
  • C 正确:X、Y 独立 ⇒ D(X+Y) = DX + DY = 1 + 2 = 3
  • D 错:D(XY) 公式复杂,需用 D(XY) = E(XY)² − [E(XY)]²,需要更多信息(比如 EX²、EY²),不能直接得 2
  • 答案:BC(多选)

  • 强调"独立性的作用":让 E(XY)=EX·EY 和 D(X+Y)=DX+DY 这两个简化成立
  • 提醒:期望线性 EX+EY 永远对,方差加和性 DX+DY 仅对独立(或不相关)成立
