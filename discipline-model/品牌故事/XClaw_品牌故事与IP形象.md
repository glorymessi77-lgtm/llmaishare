# XClaw · 品牌故事与 IP 形象

> AI教学产品部 · 2026-03-27

---

## 一、名字从哪来

**X** — 三重含义，一个字母承载三层故事：

| 含义 | 来源 | 映射 |
|------|------|------|
| **X 战警** | Professor X 坐在轮椅上，却能感知和调度每一位变种人的超能力 | XClaw 是教学平台的"X教授"——它自己不上课，但它调度所有AI能力为教学服务 |
| **小雅（Xiaoya）** | 讯飞高教教学平台"小雅"的首字母 | XClaw 是小雅的"智能大脑"，让小雅从工具进化为智慧体 |
| **未知数 X** | 数学中 X 代表无限可能 | 每位教师用 XClaw 创造的智能体，都是一个新的 X——未被定义的教学可能性 |

**Claw** — 爪，不是武器，是触达：

不是金刚狼的利爪用来战斗，而是章鱼的触手用来感知、抓取、协调。八条触手，独立思考，协同行动——一个中枢，同时触达教学的每一个场景。

**XClaw = X教授的智慧 × 章鱼的触手 × 无限的可能**

## 二、品牌故事

### 「给小雅一个大脑」

小雅已经陪伴高校师生很多年了。

她帮老师发课件、管成绩、收作业。她很勤快，但她只是一个工具——老师点哪里，她做哪里。老师不点，她就安静地等着。

直到有一天，小雅有了 XClaw。

XClaw 不是一个新按钮，不是一个新页面。它是小雅的大脑。

从那天起，事情变了：

> 刘老师说了一句"帮我准备明天第三章的课"，XClaw 听懂了。
> 它翻开了刘老师上传过的教材，找到了第三章的知识点，调动了备课助手生成教案，又叫来了组卷助手准备了一套随堂测验——三分钟，刘老师打开手机时，一切已经准备好了。
>
> 刘老师惊讶地看着屏幕，喃喃自语："小雅……你什么时候变得这么聪明了？"

小雅没有变。变的是她背后的 XClaw。

**XClaw 像 X 教授一样，坐在幕后，感知每一位教师的需求，调度每一个 AI 能力，让小雅从"会听话的工具"变成了"会思考的伙伴"。**

而更让人惊喜的是——

> 王老师说："我想要一个法学案例分析助手。"
> XClaw 三分钟就帮她创建了一个。不需要写代码，不需要懂AI。
> 一句话，一个专属于她的教学智能体就诞生了。

王老师给它取名叫"小法"。隔壁的陈老师听说了，也做了一个"化学实验指导员"。
再后来，全校200多位老师，每个人都有了自己的AI助手。

**XClaw 给了每一位老师一种超能力——用一句话创造自己的 AI。**

就像 X 战警里，每个变种人都有独特的能力。XClaw 让每一位老师，都成为了自己学科领域的超级英雄。

## 三、IP 形象：小X（xiǎo X）

### 3.1 形象设定

**小X** 是一只赛博朋克风格的小章鱼，但不是那种深海里阴暗的章鱼——它圆润、明亮、充满好奇心，像一个聪明的小助手。

| 特征 | 设计说明 |
|------|---------|
| **头部** | 圆润的章鱼头，额头有一个发光的 **X** 标记（像X教授的脑波标志），代表智慧中枢 |
| **眼睛** | 大而温暖的眼睛，不是冰冷的AI感，而是充满好奇和善意的表情 |
| **触手** | 8条触手，每条末端有不同颜色的微光，代表不同的能力模块 |
| **颜色** | 主体：温暖的科技蓝（#2563eb）渐变到活力橙（#f59e0b），不是冷冰冰的深蓝 |
| **配饰** | 头顶一个小小的学士帽（微微歪着），暗示教育属性，又带一点俏皮 |
| **大小** | 设计上偏 Q 版，约 3-4 头身比，适合做产品 icon、加载动画、引导角色 |

### 3.2 八条触手 = 八大能力

每条触手代表 XClaw 的一种核心能力，末端发光颜色不同：

| 触手 | 能力 | 颜色 | 场景 |
|------|------|------|------|
| 🔵 ① | 智能备课 | 蓝色 | "帮我准备明天的课" |
| 🟢 ② | 智能批改 | 绿色 | "批改这50份作业" |
| 🟡 ③ | 学情分析 | 金色 | "哪些学生在掉队？" |
| 🟣 ④ | 智能体创编 | 紫色 | "帮我建一个法学助手" |
| 🟠 ⑤ | 知识库 | 橙色 | "从我的教材里找答案" |
| 🔴 ⑥ | AI问答 | 红色 | "学生的问题24小时解答" |
| 🩵 ⑦ | 多模型调度 | 青色 | 自动选择最优AI模型 |
| ⚪ ⑧ | 开放连接 | 白色 | 对接现有教学平台 |

### 3.3 表情系统（产品中的微交互）

小X 不是静态的 logo，它在产品中是有"情绪"的：

| 状态 | 表情 | 出现场景 |
|------|------|---------|
| **待命** | 眨眨眼，触手轻轻摆动 | 等待用户输入时 |
| **思考中** | 触手缩起来抱住头，额头X标记闪烁 | AI正在处理任务时 |
| **完成了** | 触手张开，竖起一条触手比"OK" | 任务完成时 |
| **出错了** | 歪头，触手挠后脑勺，表情不好意思 | 遇到错误时 |
| **发现了** | 眼睛放大发光，一条触手举起灯泡 | 给出建议或发现问题时 |
| **多任务** | 多条触手同时忙碌，每条做不同的事 | 并行处理多个任务时 |

### 3.4 小X 的性格设定

> **"我不是替你工作的AI，我是帮你变强的伙伴。"**

- **聪明但不傲慢**：它知道答案，但更喜欢引导你思考
- **勤快但不越界**：它会主动提醒你"学情有异常"，但不会替你做决策
- **温暖但不黏人**：完成任务就安静退下，不会刷存在感
- **诚实但不冰冷**：做不到的事会直说"这个我还不会，但我在学"

## 五、品牌视觉关键词

| 维度 | 关键词 | 避免 |
|------|--------|------|
| **色彩** | 科技蓝+活力橙+温暖白 | 冷灰、纯黑、过于炫酷的霓虹 |
| **风格** | 温暖科技感、亲和专业 | 冰冷机械感、过于卡通幼稚 |
| **字体** | 圆润现代（中文用思源黑体/鸿蒙字体） | 棱角分明、过于严肃的宋体 |
| **调性** | 聪明的伙伴、可信赖的助手 | 万能的AI、冰冷的机器人 |
| **核心感受** | "这个AI懂教学" | "又一个通用AI工具" |
