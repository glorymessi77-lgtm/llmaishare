CHAPTER 01

核心场景 · 教学4场景 + 论文3场景

场景 ≠ 建库动作。场景是教师/学生日常反复做的事，建库是一次性的基础设施搭建。教师早上起来不会想"今天我要建知识库"——他想的是"明天的课还没备"。

教学场景 · T1 · T4 · T2 · T3

T1 · 智能问答+溯源 P0
教师和学生基于知识库进行学科专业问答。每个回答溯源到教材具体页码段落，知识库中没有的内容不编造，明确拒答并引导查阅教材。
INPUT
"SN1和SN2反应机理有什么区别？"
"第三章习题5怎么做？"
OUTPUT
结构化回答 + 溯源标签（教材P.186-192）+ 置信度98%
T4 · AI备课 P0
输入知识点或学习问题，基于知识库生成交互式课件——PPT（自动排版+知识点导航）、音频讲解、交互式测试题。对标OpenMac。
INPUT
"亲核取代反应SN1和SN2，90分钟"
"学生总分不清SN1和SN2"
OUTPUT
12页交互式PPT + AI音频讲解 + 嵌入式测试题 + 溯源标注
T2 · AI建课 P1
教师为课程构建知识图谱——可导入已有图谱或从教材自动生成。支撑图谱问答和自适应学习路径推荐。建课持续进行，图谱动态更新。
INPUT
教材/课件上传 + 教学大纲
已有图谱导入 + 学生掌握度数据
OUTPUT
课程知识图谱 + 图谱问答 + 自适应学习路径 + 图谱更新通知
T3 · 智能出题 P1
教师指定知识点范围、题型、数量和难度，系统从题库检索+AI生成新题，自动组卷。答案经校验后正确率必须100%。
INPUT
"从第6章亲核取代出10道选择题，中等难度"
"期末试卷：选择20+简答5+计算3"
OUTPUT
格式化试卷 + 标准答案+解析 + 知识点+难度标注

论文场景 · R1 · R2 · R3

R1 · 论文检索+摘要 P0
"找不对称催化相关的最新综述，给我摘要和核心结论"——高频使用，每周多次。直接驱动论文入库。
INPUT
研究方向/关键词/时间范围/期刊范围
OUTPUT
论文列表 + AI摘要 + 引用关系标注 + 一键导入参考文献
R2 · 科研选题探索 P1
"碳正离子领域还有什么方向没人做？"——孙校+何老师强烈建议加入。"老师做科研最缺idea"。
INPUT
研究方向关键词 + 排除已知方向 + 时间范围
OUTPUT
研究方向热力图 + 研究空白列表 + 脉络演进图 + 推荐选题
R3 · 科研写作辅助 P1
"帮我写国基申报书的研究综述部分"——孙校+何老师强烈建议加入。含引用标注、引文格式、数据分析辅助。
INPUT
写作任务类型 + 大纲/要求 + 参考文献列表
OUTPUT
写作初稿（含引文）+ 格式化参考文献 + 数据分析描述 + 插图建议

砍掉/降级的场景

  • 批改（P2）：OCR手写+评分一致性，技术风险高
  • 督导（P2）：ASR转写+教学行为分析，频率低
  • 学情分析（P2）：需要使用数据积累，一期没数据
  • 科研知识沉淀：行为积累库自然覆盖此需求——使用检索/选题/写作的过程中，研究足迹自动沉淀
  • 投稿建议（P2）：需期刊匹配数据库，属于增值功能
